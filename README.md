# Banking-Management-System-Python-

A simple console-based Banking Management System built using Python and Object-Oriented Programming (OOP).
This project simulates basic banking operations like account creation, deposit, withdrawal, updating details, and deleting accounts using JSON file storage.

Features
🔹 Create New Account
🔹 Secure Login using Account Number & 4-digit PIN
🔹 Deposit Money (with validation)
🔹 Withdraw Money (with balance check)
🔹 Update Account Details
🔹 Delete Account
🔹 Persistent Data Storage using JSON
🔹 Input Validation & Error Handling

Technologies Used
🔹Python 3
🔹Object-Oriented Programming (OOP)
🔹JSON
🔹File Handling
🔹Exception Handling

How It Works
🔹User data is stored in a data.json file.
  Each account includes:
    🔹Name
    🔹Email
    🔹Gender
    🔹Age
    🔹PIN
    🔹Account Number
    🔹Balance

The system validates:
🔹Age must be 18 or above
🔹PIN must be exactly 4 digits
🔹Deposit amount must be valid
🔹Withdrawal only if sufficient balance
